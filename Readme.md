# Quartz Scheduler

- A simple crud contact app which filters contacts and sends email to any contact with a birthday.

## Technologies

- Java 8+
- Spring (Boot, Security, Data JPA)
- Hibernate
- PostgreSQL
- Lombok
- JUnit, Mockito
